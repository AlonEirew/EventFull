Overview

You'll see sentences with events sharing an equal time relation (events that started at the same time). You need to decide IF these events refer to the same event in the world.
For example,

[A traveler is kidnapped, and the police officers said he was taken two days ago.]

[QUESTION] Which of the highlighted events refers to the same (kidnapped) event?

EXPLANATION: Here, we have only a single event mention candidate that has equal start time with the event mention in focus. We need to determine if the event mentions (kidnapped) and (taken) describe the same real-world event. To make this decision, we must consider the arguments associated with both event mentionsâ€”the time, location, and participants. We observe that both mentions share the same participants (the kidnapped individual, "traveler/he") and the same time frame ("two days ago"), confirming that they indeed represent the same event.

IMPORTANT RULE

An event is usually associated with four components: 1) an action component referring to what happens or holds; 2) a time slot which is responsible for anchoring the action in time ; 3) a location component which links the action component to a place/location; and 4) a participant component, which illustrates the "who" or "what" is involved in the action component.

We can say that two events are the same event if and only if we can infer that they share the same action, time, location, and participant(s), as much as indicated in the text and using common-sense.

[Achilles was running in the left lane and the turtle was running in the right lane]

[QUESTION] Which of the highlighted events refers to the same (running) event?

EXPLANATION: while both (running) and (running) share the same time and action, they are clearly referring to different participant (Achilles and the turtle), therefore we can deduce they are two different events.
