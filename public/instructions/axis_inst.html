Overview

We want to find out which event can be anchored in time, i.e., has a well-defined notion of time. You will be given a question in the following format:

[In Washington today, the Federal Aviation Administration (released) air traffic control tapes from the night the TWA Flight eight hundred went down.]

Can the event (released) be anchored in time?

  • Yes. It's anchorable (You need to choose this)
  • No. It's intention/wish/opinion
  • No. It's hypothetical/condition
  • No. It's negation
  • No. It's abstract/non-specific
  • No. It's static
  • No. It's recurrent

You are able to select ONLY one of them, but if you think multiple choices are correct, please choose the first one from top to bottom. For example, if an event is both "intention" and "negation", you only need to choose "intention".

What can be anchored

We say an event can be anchored in real timeline if and only if

  • Case 1: Event happened in the past, or is happening now.
  • Case 2: Event is unconditionally planned to happen in future.

For example:

You should answer "Yes" to the following situations:

  • Case 1
    • In Washington today, the Federal Aviation Administration (released) air traffic control tapes from the night the TWA Flight eight hundred went down.
  • Case 2
    • I will be (leaving) tomorrow.
