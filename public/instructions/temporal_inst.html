Overview

You'll see sentences with two events marked with different colors and you need to decide the temporal order taking into consideration only the starting time of the events. For example,

[A traveler is kidnapped, and the police officers said the kidnapper is demanding money.]

[QUESTION] Which event started first?

  • kidnapped (You need to choose this)
  • said
  • Both started at the same time
  • Uncertain

EXPLANATION: (kidnapped) DEFINITELY started before (said) if we consider the chain of events. The 2nd answer is obviously NOT POSSIBLE. The 3nd answer might be possible if considering that kidnapped event is still ongoing. However, even if an event is durative, e.g., (kidnapped) is during a period of time, you only need to consider the STARTING time (when the event began/started) to make the decision.

Not all cases are a clear-cut as (kidnapped) and (said). For example,

[I ate a burger and drank a bottle of water for lunch today.]

[QUESTION] Which event started first?

  • ate
  • drank
  • Both started at the same time
  • Uncertain (You need to choose this)

EXPLANATION: Here we only know that (ate) and (drank) happened during lunchtime today. Based on the text, we cannot rule out the possibility that (ate) is earlier than (drank), and we cannot rule out the possibility that (drank) is earlier than (ate), either.

IMPORTANT RULE 2

Two events may have exactly the same start time, although it's not quite often. For example,

[They (filed) objections to the court, (claiming) that the suspects were treated unfairly.]

[QUESTION] Which event started first?

  • filed
  • claiming
  • Both started at the same time (You need to choose this)
  • Uncertain

EXPLANATION: (filed) and (claiming) are the same thing, so their start times are equal: neither before nor after.

RECOMMENDED STRATEGY

For each question, try to place both events on a timeline considering only the starting time of the events. (1) Try to find explicit indicators between the events (before that, after, simultaneously, etc..), (2) use common sense (like in the above kidnapped example), and (3) In case common-sense is needed, try to determine the events order, test if changing the events order still makes sense, in case it does then its inconclusive/uncertain.

Take the above questions for example. First, assume (kidnapped) indeed starts before (said) starts. Then ask yourself if it makes sense. We know from the sentence that the police are working on the kidnap, so based on common sense, the kidnap must start earlier so that the police can work on it. Any other way is logically impossible.

In the (ate) and (drank) example, first assume (ate) starts before (drank) starts, does it makes sense? Then try to replace the order so now (drank) starts before (ate), does it make sense? Since both cases makes sense, we cannot determine which one started first inconclusively.
